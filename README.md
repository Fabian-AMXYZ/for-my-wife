# A Personal Project
A personal showcase of letter for my significant other

## Features
- **Interactive Homepage:** A "Selection Menu" to choose which month's memory to view.
- **Dynamic Letter System:** Smoothly switches between different letters without reloading the page.
- **Cutesy Aesthetic:** Designed with soft pinks, custom bubbly fonts, and "sticker-style" UI elements.
- **Mobile Optimized:** Fully responsive for viewing on smartphones (perfect for a surprise link!).

##Photography Focus
As time goes on, it will be not only letters but showcase of how I improve on taking pictures for her

## 🛠️ Built With
- **HTML5** & **CSS3** (Flexbox/Grid, Custom Animations)
- **JavaScript** (Vanilla - for dynamic content and navigation)
- **Google Fonts** (Gaegu & Quicksand)
